# AO Reading

AO 创伤骨科教材精读与微信公众号发布项目。

本仓库用于整理「精读 AO」栏目内容，从英文原文摘录、中文翻译、重点词汇整理，到英文朗读文本和微信公众号文章草稿，形成稳定、可复用的内容生产流程。

## 目录结构

```text
AO-Reading/
├── AO-Principles.md
├── Original_Texts/
├── Translations/
├── Vocabulary/
├── ElevenLabs/
├── WeChat_Articles/
└── Templates/
```

## 目录用途

- `AO-Principles.md`：AO 原则相关笔记与后续整理内容。
- `Original_Texts/`：存放 AO 英文原文摘录。
- `Translations/`：存放英文原文对应的中文翻译。
- `Vocabulary/`：存放重要及常见词汇、短语、术语解释。
- `ElevenLabs/`：存放为 ElevenLabs 生成的英文朗读文本。
- `WeChat_Articles/`：存放最终可发布到微信公众号的文章草稿。
- `Templates/`：存放栏目输出模板和写作格式规范。

## 固定工作流程

1. 列出英文原文。
2. 翻译原文。
3. 挑选重要及常见词汇。
4. 为 ElevenLabs 生成英文朗读文本。
5. 整理成微信公众号可发布的文章草稿。
